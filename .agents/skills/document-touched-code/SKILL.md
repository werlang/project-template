# Document Touched Code

Use this skill for code-writing tasks.

## Rules

- Add or update JSDoc for touched exported classes, methods, and functions.
- Keep comments focused on intent, assumptions, invariants, and non-obvious ordering.
- Remove stale comments when behavior changes.
- Do not add comments that only restate the next line of code.
- Keep documentation aligned with the actual implementation.
